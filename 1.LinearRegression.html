What is regression ?

Dependend Variale & Independent Variable

Regression is a technique by which we can predict the value of Dependent on the basis of Independent Variable.

Linear Regression : when dependent variable and independent variable have a linear relationship we can use line equation to predict dependent variable's value using indepent variable and this is known as linear regression.

$$ y = f(x) $$$$ y = \beta_0 + \beta_1 X $$

where $\beta_0$ is intercept, $\beta_1$ is coefficient/slope, y is label and X is features

$$ Error = | y_\text{actual} - y_\text{pred}| $$

Residual Sum of Squares $$ RSS = (y_\text{actual} - y_\text{pred} )^2 $$

$$ RSS = \sum\limits_{i=0}^N (y_i - \hat y_i)^2 $$
$$ RSS = \sum\limits_{i=0}^N (y_i - (\hat\beta_0 + \hat\beta_1 X_i))^2 $$
$$ \hat \beta_1 = \frac {\sum\limits_{i=0}^N (y_i-\bar y)(x_i - \bar x)} {\sum\limits_{i=0}^N(x_i-\bar x)^2} $$$$ \hat\beta_0 = \bar y - \hat\beta_1 \bar x $$

MSE stands for Mean Squared Error $$ MSE = \frac { \sum\limits_{i=0}^N(y_i - \hat y_i )^2} {N} $$

RMSE stands for Root Mean Squared Error

$$ RMSE = \sqrt { \frac { \sum\limits_{i=0}^N(y_i - \hat y_i )^2} {N} } $$$$ R^2\text{ Score} = \frac {TSS-RSS} {TSS} = 1 - \frac {RSS} {TSS} $$

def split(features, target, test_size=0.3):
    """
        will split data into training and testing part according to testsize
    """
    total_size = target.size
    test_size = int(np.ceil((total_size * test_size)))
    train_size = total_size - test_size
    total_indexes = [ index for index in range(0,total_size)]
    X_train = []
    y_train  = []
    for var in range(train_size):
        index = np.random.choice(total_indexes)
        total_indexes.remove(index)
        X_train.append(features[index])
        y_train.append(target[index])
    X_test = [ features[index] for index in total_indexes ]
    y_test = [ target[index] for index in total_indexes ]
    return np.array(X_train), np.array(X_test), np.array(y_train), np.array(y_test)

def linear_regression(x, y):
    """
        a linear model which will return intercept and coefficient at minimum RSS
    """
    x_bar = x.mean()
    y_bar = y.mean()
    b_1 = ((x-x_bar)*(y-y_bar)).sum() / ((x-x_bar)**2).sum()
    b_0 = y_bar - (b_1 * x_bar)
    return b_0, b_1

def RSS(y, y_pred):
    return ((y-y_pred)**2).sum()

def TSS(y):
    return ( (y-y.mean())**2).sum()

def MSE(y, y_pred):
    return RSS(y,y_pred)/y.size

def RMSE(y,y_pred):
    return np.sqrt(MSE(y, y_pred))

def R2_score(y, y_pred):
    rss = RSS(y, y_pred)
    tss = TSS(y)
    return 1 - (rss/tss)
